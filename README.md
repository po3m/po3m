# Po3m

**Original poetry by Dr Gi Chao, in collaboration with Donutree.**

🌐 **Live site:** [po3m.com](https://po3m.com)

---

## About

Po3m is a collaborative poetry project exploring inheritance, memory, and the body through verse. The poems are written by [Dr Gi Chao](https://gichao.com), with assistance from Donutree (an AI collaborator).

The project aims to publish 100 poems toward a book, written in public, rendered with code.

## Features

- **Shader backgrounds** — each poem rendered with animated WebGL visuals
- **Open submissions** — readers can submit response poems
- **Moderated comments** — conversation around the work
- **CC BY-NC 4.0** — free to share with attribution, non-commercial

## Tech Stack

- Cloudflare Pages + Workers
- D1 (SQLite) database
- Vanilla JS + WebGL shaders
- No framework, minimal dependencies
